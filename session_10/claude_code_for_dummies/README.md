# Claude Code For Dummies

*The clinical AI builder's guide to shipping software with Claude as your pair programmer*

---

**About this book.** This is a community-style "For Dummies" companion for Claude Code, written for clinicians, data scientists, and AI builders at King Hussein Cancer Center (and anyone else who wants to learn). It uses clinical and AIDI examples throughout because that's the readership — but every pattern works on any codebase.

**Audience.** Doctors, pharmacists, nurses, biostatisticians, and clinical informaticists who can write a little Python or R but have never run a coding agent. No CS degree required.

**Length.** ~40,000 words across 22 short chapters in 5 parts, plus a cheat sheet and two appendices.
