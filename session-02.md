# Session 2: Python Basics + GitHub

Key topics:
- Python fundamentals, API calls, version control, repos, branching, pull requests.

## Instructions
- Learn core Python syntax: variables, loops, functions, and error handling.
- Learn basic GitHub workflow: create repo, commit changes, and open pull requests.

## Practice
- Create a small Python script that sends one LLM request and prints the response.
- Push the script to a new GitHub repository with clear commit messages.
