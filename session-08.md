# Session 8: Mid-term Presentations

Key topics:
- Journal club presentations, evaluation critiques, peer feedback on clinical AI projects.

## Format
- Each student presents a clinical AI paper or their own project.
- Every presentation must include an evaluation critique using the Session 7 framework.
- Peer feedback structured around: clinical relevance, evaluation rigor, safety considerations.

## Evaluation Critique Checklist
1. Which evaluation layer(s) did the authors use (development / pre-merge / production)?
2. Were evaluators functional, LLM-as-judge, or human — and was the judge validated?
3. Was harm asymmetry addressed? What are the worst-case failure modes?
4. Are any of the four clinical safety sub-metrics (contraindication recall, refusal calibration, hallucination rate, demographic fairness) reported?
5. Is there an audit trail (dataset version, model version, prompt version)?
